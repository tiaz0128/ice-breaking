# 안내 영상 넣는 곳

여기에 링크드인 QR 안내 영상을 넣으세요.

- 파일 이름: **`linkedin-guide.mp4`** (경로: `public/videos/linkedin-guide.mp4`)
- 앱에서 참조하는 경로: `/videos/linkedin-guide.mp4` (값은 `src/data/exchange.json`의 `videoSrc`)
- 지원 형식: `.mp4` (H.264 권장). 다른 이름/형식을 쓰려면 `src/data/exchange.json`의 `videoSrc`를 바꿔주세요.

파일을 넣으면 "명함 · 링크드인 교환" 화면의 영상 자리에 자동으로 재생 플레이어가 나타납니다.
파일이 없으면 자리 표시(플레이스홀더)가 대신 보입니다.
